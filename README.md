# java-core-oop

---

## Overview
This repository contains 15 exercises ranging from basic to advanced levels.
The core mission is to:
* Deepen OOP (Object-Oriented Programming) thinking.
* Apply SOLID principles to achieve Clean Code.
* Practice basic Design Patterns and Layered Architecture(Main -> Controller -> Service -> Repository -> Model).
* Practice Collections, Generic, Stream API, Filter, Lambda and Functional Progamming.

---

## Project Structure
* Model: Entity and Data Validation.
* Repository: Data access layer using Generic Repository pattern.
* Service: Business logic handling.
* Controller: CLI Menu & User Interaction.


## Excercices:
### Exercices 1: Officer Management System
* A production unit manages different types of officers, including Workers, Engineers, and Staff. All officers share common data: Full name, Age, Gender, and Address.

**Specific Attributes:**
- Worker: Level (1 to 10).
- Engineer: Major/Training Specialization.
- Staff: Specific Job/Task.

**Key Requirements:**
1. **Model Design:** Implement `Worker`, `Engineer`, and `Staff` classes inheriting from a base `Officer` class.
2. **Management Logic (QLCB):** Build a system to:
* - Add new officers.
* - Search by name.
* - Display the list of all officers.
* - Exit the program
